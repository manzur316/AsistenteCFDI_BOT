## Conceptos utiles para proponer

- CCTV / videovigilancia: requiere `c_ClaveProdServ` oficial local para proponer nuevas claves.

- Las claves de Compact no activan conceptos por si solas.

## Advertencias

- No se encontro catalogo oficial local `c_ClaveProdServ`; todas las claves Compact quedan pendientes de confirmacion oficial.
- No se copio texto extenso del PDF.
- No se uso OCR masivo.
